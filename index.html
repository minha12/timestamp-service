Timestamp Microservice API Documentation

GET /api/timestamp/:date_string?

Returns the Unix timestamp and UTC string for a given date.

Demo

Results will appear here...

Parameters

Name Type Description
date_string string Optional. Date in ISO-8601 format or Unix timestamp

Example 1: Valid ISO Date

GET /api/timestamp/2015-12-25

Response:

{
  "unix": 1451001600000,
  "utc": "Fri, 25 Dec 2015 00:00:00 GMT"
}

Example 2: Unix Timestamp

GET /api/timestamp/1451001600000

Response:

{
  "unix": 1451001600000,
  "utc": "Fri, 25 Dec 2015 00:00:00 GMT"
}

Example 3: Current Time

GET /api/timestamp

Response:

{
  "unix": 1683900000000,
  "utc": "Fri, 12 May 2023 12:00:00 GMT"
}

Example 4: Invalid Date

GET /api/timestamp/invalid-date

Response:

{
  "unix": null,
  "utc": "Invalid Date"
}
